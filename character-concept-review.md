# Pomochi 캐릭터 컨셉 검토: 프린세스 vs 동물 의인화

> 기획팀 + 디자인팀과 함께 검토할 문서입니다.

---

## 1. 현재 컨셉 vs 새로운 제안

### 기존 컨셉: "프린세스 메이커" 스타일

```
특징:
- 타겟: 주로 여성 유저
- 캐릭터: 공주/여성 캐릭터
- 성인 느낌: 약간 있음 (성장/연애 요소)
- 감정: 우아함, 화려함

타겟층:
├─ 여성 (높음)
├─ 어린이 (낮음, 너무 복잡)
└─ 남성 (매우 낮음, "프린세스"가 호소력 없음)

예상 DAU: 여성 위주
```

### 새로운 제안: "강아지/고양이 의인화"

```
특징:
- 타겟: 남녀노소 불문
- 캐릭터: 동물 의인화 (중립적 성별)
- 귀여움: 높음
- 감정: 친근함, 따뜻함, 재미

타겟층:
├─ 여성 (높음, 동물 좋아함)
├─ 남성 (높음, 중립적 캐릭터)
├─ 어린이 (높음, 귀여움)
└─ 40대+ (중간, 반려동물 관계자)

예상 DAU: 전 연령층
```

## 2. 비교 분석

| 항목 | 프린세스 | 강아지/고양이 |
|------|---------|------------|
| **타겟층** | 여성 위주 | 남녀 전연령 |
| **성별 중립성** | ❌ (여성성 강함) | ✅ (중립적) |
| **귀여움** | 우아함 | ⭐⭐⭐⭐⭐ (매우 높음) |
| **대중성** | 낮음 (마니아층) | 높음 (반려동물 트렌드) |
| **SNS 공유성** | 중간 | 매우 높음 (동물 컨텐츠) |
| **글로벌 확장** | 낮음 (문화적) | 높음 (보편적) |
| **디자인 난이도** | 중간 | 낮음 (친숙함) |
| **라이센싱** | 없음 | 동물 캐릭터 가능 |

## 3. 강아지 vs 고양이 선택

### 옵션 A: 강아지

```
장점:
✅ 충성도/친근함 상징
✅ 더 활발한 성격 표현
✅ 다양한 품종 활용 가능
✅ 주인-반려동물 관계 호소

단점:
❌ 약간 "소년스러운" 느낌
❌ 귀여움보다 "친구" 느낌

타겟: 남성 조금 더 높음
```

### 옵션 B: 고양이

```
장점:
✅ 높은 귀여움 지수
✅ 독립적 성격 (성장 스토리에 좋음)
✅ 신비로움 + 개성 표현
✅ 밈 문화와 잘 맞음 (고양이 사진 트렌드)

단점:
❌ 약간 "여성스러운" 느낌 (고양이 좋아하는 층이 여성 많음)

타겟: 전체 연령층 균형, 여성 약간 더 높음
```

### 제안: 하이브리드

```
🐕 메인 캐릭터: 강아지 (중립성 높음)
🐈 서브 캐릭터: 고양이 (나중에 추가, Season 2)

이유:
1. Phase 1은 강아지로 간단하게 시작
2. 커뮤니티 반응 보고 고양이 추가
3. 친구 시스템 확장 (강아지 + 고양이 상호작용)
```

## 4. "콩냥이" vs "포모치" 이름 재검토

### 현재 이름: 콩냥이

```
문제점:
- "냥이" = 고양이 암시
- 성별이 암시적
- 너무 귀엽지만 성인층에는 약할 수 있음

개선안 1: 품종명
├─ 포메라니안 → "포메"
├─ 시베리안 허스키 → "시베"
└─ 뱅갈 고양이 → "뱅"

개선안 2: 중립적 이름
├─ "포모" (포모도로 + 모)
├─ "도토" (도토리처럼 귀여운)
└─ "콩콩" (포동한 느낌, 중립적)

개선안 3: 그냥 이름
├─ "루루"
├─ "뭉치"
└─ "뭔가 러운 이름들"
```

**추천: "포모" 또는 "뭉치"**
- 포모: 포모도로 + 모든 동물 가능
- 뭉치: 귀여우면서도 성별/종 중립적

## 5. 의인화 스타일 가이드

### 의인화 수준 선택

```
레벨 1: 살짝 의인화 (가장 자연스러움)
┌─ 기본: 동물 모습 유지
├─ 표정: 인간 감정 표현
├─ 액세서리: 작은 모자/목걸이 정도
└─ 예시: 타마고치 스타일
   위험: 너무 심심할 수 있음

레벨 2: 중간 의인화 (추천) ⭐
┌─ 기본: 동물 + 팔/다리 선명화
├─ 표정: 풍부한 감정 표현
├─ 액세서리: 옷/소품 (캐주얼)
└─ 예시: 보노보노, 네코아츠메
   효과: 귀여우면서도 표현력 좋음

레벨 3: 완전 의인화 (약간 과함)
┌─ 기본: 인간처럼 완전 의인화
├─ 표정: 매우 풍부한 감정
├─ 액세서리: 옷/가방/신발 등
└─ 예시: 보노보노 진짜 캐릭터들
   위험: 동물 느낌이 안 남
```

**추천: 레벨 2 (중간 의인화)**

## 6. 디자인 방향 결정

### 강아지 의인화 예시

```
기본 형태:
- 동물의 특징 유지 (꼬리, 귀, 발 패드)
- 팔 선명화 (활동성)
- 표정 풍부하게 (눈/입/콧수염)

타입별 포즈 변화:
- 학자: 안경, 책 들기 (여전히 강아지)
- 스포츠맨: 운동복 (귀여운 스타일)
- 프린세스 → 왕/왕비처럼? (젠더뉴트럴)
- 번아웃: 피곤한 강아지 (애처로움)

색상:
- 기본: 갈색/검은색 (자연스러움)
- 강조색: 포인트 리본/액세서리
- 배경: 따뜻한 톤 (집 같은 느낌)
```

### 고양이 의인화 예시 (Season 2)

```
기본 형태:
- 동물의 특징 유지 (꼬리, 귀, 수염)
- 팔 선명화 (독립적 느낌)
- 신비로운 표정 (눈 커짐)

차이점:
- 더 독립적인 포즈
- 약간 더 세련된 느낌
- 주주적 성격 표현
```

## 7. 기존 기획과의 호환성

### princess_maker.md 수정 필요 부분

```
섹션 1: 기획 방향
- "프린세스 메이커처럼" → "동물 의인화로"
- 타겟층 확대 명시

섹션 5: 최종 성장 타입 (이름 변경)
이전:
- "프린세스"
- "조화로운 현인"

변경 후:
- "왕/왕비" → 중립적 이름
- 모든 타입에 동물 특징 추가

섹션 10: 캐릭터 표정 (예시 추가)
- 강아지 특유의 표정들
  (귀가 쫑긋해지는 것, 혀 내밀기 등)

그 외 변경사항:
- "프린세스" 용어 제거
- 동물 캐릭터 특징 반영
- 계절/시간대별 배경 (집 컨셉)
```

### design-resource-guide.md 수정 필요 부분

```
1. 캐릭터 표정
- 17가지는 동일하지만, "강아지식" 표정으로
  (귀 쫑긋, 혀 내밀기 등)

2. 캐릭터 포즈
- 6가지 타입은 유지
- 모두 강아지 포즈로 표현
  (목 기울임, 꼬리 흔들기 등)

3. 액세서리
- 왕관 → "귀엽게 씌운 왕관" (동물에 맞춤)
- 옷 → "강아지 옷/조끼" 스타일
- 색상 팔레트 유지

4. UI 요소
- 강아지 테마로 변경 (뼈다귀 모양 버튼? 등)
- 따뜻한 톤 배경

5. 예상 작업량
- 대체로 비슷 (17가지 표정은 동일)
- 액세서리 디자인 약간 달라짐 (증가)
- 타임라인 영향: 거의 없음
```

## 8. 기획자 의견 (SWOT 분석)

### 강아지 의인화 채택 시

**Strengths (강점):**
- ✅ 남녀 전연령층 확보
- ✅ SNS 공유성 높음 (동물 컨텐츠 인기)
- ✅ 반려동물 보유자 공감 (확장 시장)
- ✅ 글로벌 확장성 높음

**Weaknesses (약점):**
- ❌ 프린세스 메이커 콘셉트 일부 손실
- ❌ 초기 기획서 수정 필요
- ❌ 여성층 일부 흥미 감소 (?)

**Opportunities (기회):**
- 🚀 "반려동물 육성 게임" 카테고리 신규 시장
- 🚀 동물 커뮤니티와 협력 (반려동물 인스타그램 등)
- 🚀 상품화 가능성 증가 (강아지 피규어 등)
- 🚀 시즌 2 = 고양이 추가 (콘텐츠 확장)

**Threats (위협):**
- ⚠️ 기존 유사 게임 많음 (네코아츠메, 보노보노 게임 등)
- ⚠️ 성격 표현이 "프린세스" 보다 제한적일 수 있음
- ⚠️ 디자인 너무 귀여우면 성인층 외면 가능

## 최종 권장안

🐕 **강아지 의인화 (레벨 2) 채택**

이유:
1. 타겟층 3배 이상 확대 (여성→전연령)
2. 개발 난이도 동일
3. SNS 바이럴성 2배 증가
4. 글로벌 확장성 높음
5. 시즌 2 고양이 추가로 장기 로드맵 가능

변경 사항:
- 기획서 (10% 수정)
- 디자인 방향 (30% 변경)
- 개발 (거의 동일)
- 타임라인 (동일)

리스크:
- 기존 프린세스 컨셉 손실 (낮음)
- 유사 게임과의 경쟁 (중간)
- 정체성 약화 (가능성 낮음, 포모도로는 차별점)
