## 1. 디자인 리소스 카테고리

### Category A: 캐릭터 표정 (Phase 3-1에서 필요)

**필요한 표정:**
```
1. 기본 상태 (일일 플레이)
   - 행복 (눈이 반짝임)
   - 집중 (진지한 표정)
   - 피곤 (반쯤 감긴 눈)
   - 화남 (찌푸린 눈썹)

2. 7일차 중간 평가 (4가지 상태)
   - 밝은 미소 (균형잘잡은 플레이)
   - 피곤한 눈 (극단화 경고)
   - 졸린 눈 (거의 활동 없음)
   - 하품 (휴식/잠만)

3. 14일차 결과 (타입별 표정)
   - 프린세스: 우아한 미소
   - 조화로운 현인: 지혜로운 표정
   - 학자: 차분한 표정
   - 스포츠맨: 활기찬 표정
   - 치유사: 따뜻한 표정
   - 번아웃: 지쳐있는 표정

총 17가지 표정 필요
```

**기술 사양:**
- [ ] 해상도: 400x400px? 600x600px? (협의)
- [ ] 포맷: PNG (투명배경) + SVG?
- [ ] 파일 이름 규칙: `emotion_[name]_[state].png`
- [ ] 애니메이션: 정적 이미지만? or gif/MP4?

---

### Category B: 강아지 포즈/액세서리 (Phase 4-1에서 필요)

**타입별 필요 리소스:**

```
1. 균형잡힌 왕견 ⭐⭐⭐⭐⭐
   - 기본 포즈: 우아한 강아지 자세, 앞다리 들기
   - 액세서리: 왕관 (귀에 맞춤형), 고급 목걸이
   - 특징: 꼬리 위로 펴짐
   - 색상: 갈색/검은색 + 금색 포인트

2. 창의력 많은 예술견 ⭐⭐⭐⭐
   - 기본 포즈: 팔레트와 붓 들기, 상상의 표정
   - 액세서리: 베레모, 팔레트, 붓
   - 특징: 눈이 크고 생기발랄함
   - 색상: 밝은 톤 + 다양한 색상

3. 똑똑한 학자견 ⭐⭐⭐
   - 기본 포즈: 책을 보고 있는 자세
   - 액세서리: 안경, 책 한두권
   - 특징: 집중하는 표정, 한쪽 귀가 쫑긋
   - 색상: 갈색 + 주황색

4. 활발한 탐험견 ⭐⭐⭐
   - 기본 포즈: 활기찬 자세, 가방 메기
   - 액세서리: 가방, 모자, 지도
   - 특징: 혀를 내밀고 꼬리 흔들기
   - 색상: 밝은 색상 + 빨간색

5. 따뜻한 치유견 ⭐⭐⭐
   - 기본 포즈: 포근한 자세, 친구와 함께
   - 액세서리: 꽃, 마음 모양 목걸이
   - 특징: 따뜻한 미소, 눈이 반짝임
   - 색상: 분홍색 + 주황색 포인트

6. 밤샤수견 ⭐
   - 기본 포즈: 졸린 자세, 별을 보기
   - 액세서리: 별, 달 모양
   - 특징: 반쯤 감긴 눈, 하품하는 입
   - 색상: 진한 색상 + 파란색
```

**기술 사양:**
- [ ] 캐릭터 기본 크기: 260x320px?
- [ ] 액세서리 레이어링: 분리된 파일?
- [ ] 포즈 변형: 6가지 포즈만? or 더 많이?
- [ ] 색상 변경 가능성: CSS 변수 활용? or 다중 파일?

### Category C: UI 요소 (Phase 3 전체)

**필요한 UI 요소:**

```
1. 7일차 중간 평가 화면
   - 캐릭터 배경 (4가지 상태별)
   - 말풍선 배경 이미지
   - 버튼 (계속, 다시 보기 등)
   - 아이콘 (⭐, ⚠️, ✨ 등)

2. 14일차 결과 화면
   - 배경 (축제 분위기?)
   - 등급 배지 (⭐1~5개)
   - 타입별 프레임 (6가지)
   - 통계 그래프 배경

3. 태그 버튼 (Phase 1)
   - 집중: 책/노트북 아이콘
   - 운동: 수심 아이콘
   - 휴식: 소파 아이콘
   - 잠: 침대/별 아이콘
   - 각 버튼: 기본/선택/완료 상태 3가지

4. 애니메이션
   - 표정 전환 애니메이션 (부드러운 fade?)
   - 포즈 전환 애니메이션
   - 버튼 피드백
```

**기술 사양:**
- [ ] 그래프 표현: 바 차트? 원형 차트? (협의)
- [ ] 말풍선 스타일: 카톡 스타일? or 애니메 스타일?
- [ ] 앱 테마: 라이트모드? 다크모드? 둘 다?
- [ ] 컬러 팔레트: 메인 3색 정하기

---

### Category D: 배경/이펙트 (선택사항)

```
1. 시즌 시작 화면
   - 알 깨어나는 이펙트
   - 부화 애니메이션

2. 일일 플레이 배경
   - 시간대별 배경 (아침/오후/저녁/밤)
   - 계절별 배경 변화 (선택)

3. 결과 화면 이펙트
   - 등급 공개 시 파티클
   - 캐릭터 변화 시 반짝임
```

**기술 사양:**
- [ ] 이펙트 우선순위: 필수? 선택?
- [ ] 파티클 애니메이션: Lottie? or 직접 구현?
- [ ] 배경 이미지 수: 단순화? (1개 배경으로 충분한가?)

## 6. 토론 사항

**디자인팀과 함께 결정할 것:**

1. **캐릭터 스타일**
   - 큰 눈 vs 작은 눈?
   - 귀여운 스타일 vs 우아한 스타일?
   - 선화 vs 채우기?

2. **색상 방향**
   - 따뜻한 톤 (주황/분홍) vs 차가운 톤 (파랑/보라)?
   - 원색 vs 파스텔?
   - 메인 컬러 3개 정하기

3. **애니메이션 수준**
   - 정적 이미지 (간단, 빠름) vs 복잡한 애니메이션 (예쁨, 느림)?
   - 프레임 수: 12프레임? 24프레임?

4. **배경 디자인**
   - 미니멀 (버튼만) vs 풍부한 배경?
   - 3D vs 2D?
   - 일러스트 vs 기하학적?

5. **타임라인**
   - 현실적인가?
   - 리소스(팀/인원) 확인
   - 리뷰 사이클 (몇 라운드?)

## 7. 예상 Q&A

**Q: 총 몇 개의 파일이 필요한가?**
```
A: 최소 구성:
- 표정 17개
- 포즈 6개
- 액세서리 (타입별) ~20개
- UI 아이콘 ~30개
- 배경 4개

총: ~77개 이미지
(고해상도 + 다중 포맷이면 2배 이상)
```

**Q: 한 명의 디자이너가 충분한가?**
```
A: Week 2-6 (5주) × 5일 = 25일 작업 가능
   77개 이미지 ÷ 25일 = 일일 3개 필요

   현실성: 
   - 심플 스타일: 1명 가능 (바쁨)
   - 복잡한 애니메이션: 2명 권장
   - 권장: 1명 풀타임 + 1명 파트타임
```

**Q: 수정은 몇 라운드까지?**
```
A: 제안: 2라운드
   - 라운드 1 (Week 3): 기본 표정 + 아이콘 → 피드백
   - 라운드 2 (Week 5): 전체 에셋 → 최종 확정
   
   3라운드 이상은 일정 압박
```
