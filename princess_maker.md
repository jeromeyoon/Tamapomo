# 14일 시즌제 캐릭터 성장 설계 (강아지 의인화)

## 1. 기획 방향

이 문서는 Pomochi를 단순한 타마고치형 포모도로 앱에서, 동물 의인화 캐릭터(강아지)의 성장을 중심으로 사용자의 행동에 따라 캐릭터 성장 방향이 달라지는 14일 시즌제 육성 앱으로 확장하기 위한 성장표이다.

**타겟층 확대:**
- 기존: 프린세스 메이커 (여성 위주)
- 변경: 강아지 의인화 (남녀 전연령층)

핵심은 캐릭터를 매 시즌 리셋하지 않는 것이다. 캐릭터는 계속 유지되고, 시즌은 14일 단위의 성장 챕터로 작동한다. 시즌 종료 시 사용자의 집중 습관, 집중 태그, 휴식 패턴, 중단 횟수를 바탕으로 캐릭터의 성향, 외형, 행동, 다음 시즌 보너스가 결정된다.

```text
시작: 같은 알에서 부화한 강아지 캐릭터
14일 동안: 집중 시간 + 집중 태그 + 휴식 패턴 + 방해 횟수 누적
종료: 성장 타입 결정 (6가지)
다음 시즌: 같은 강아지가 변화된 성향/외형으로 이어짐
```

목표 포지셔닝:

> 포모도로 집중과 함께 자라는 반려동물(강아지)!
> 집중 습관이 강아지의 성격과 미래를 바꾸는 게임.

## 2. 시즌 기본 구조

| 기간 | 이름 | 핵심 목적 | 열리는 변화 |
|---|---|---|---|
| 1일차 | 알 깨우기 | 첫 집중 1회 완료 | 캐릭터 부화 |
| 2-3일차 | 습관 관찰 | 어떤 태그로 집중하는지 기록 | 성향 씨앗 생성 |
| 4-5일차 | 첫 성격 형성 | 완료율과 방해 횟수 반영 | 표정 패턴 변화 |
| 6-7일차 | 미니 평가 | 중간 성장 타입 후보 2개 표시 | 작은 액세서리 획득 |
| 8-10일차 | 방향 고정 | 반복 행동에 따라 주 능력치 결정 | 행동 애니메이션 변화 |
| 11-13일차 | 성장 분기 | 스트레스와 규칙성까지 반영 | 최종 타입 후보 확정 |
| 14일차 | 시즌 엔딩 | 성장 결과 공개 | 외형 변화와 다음 시즌 보너스 |

## 3. 성장 능력치

| 능력치 | 올라가는 행동 | 너무 낮으면 |
|---|---|---|
| 집중력 | 포모도로 완료, 방해 적음 | 산만한 표정, 집중 실패 보상 감소 |
| 창의력 | 창작, 그림, 글쓰기 태그 집중 | 캐릭터 행동이 단조로움 |
| 지식 | 공부, 독서 태그 집중 | 성장 직업군 제한 |
| 체력 | 적당한 휴식, 운동 태그 | 피곤한 표정, 병 걸릴 확률 증가 |
| 감성 | 음악, 일기, 휴식 태그 | 친구 상호작용 약화 |
| 규칙성 | 비슷한 시간대 집중, 연속 출석 | 시즌 평가 흔들림 |
| 스트레스 | 과도한 집중, 휴식 부족, 중단 많음 | 마스크, 기침, 짜증 행동 |

## 4. 집중 태그별 성장 방향

| 집중 태그 | 주 능력치 | 보조 능력치 | 성장 후보 |
|---|---|---|---|
| 공부 | 지식 | 집중력 | 학자, 전략가 |
| 독서 | 지식 | 감성 | 이야기 수집가, 사서 |
| 창작 | 창의력 | 감성 | 아티스트, 디자이너 |
| 코딩/문제풀이 | 집중력 | 지식 | 연구가, 발명가 |
| 운동 | 체력 | 규칙성 | 탐험가, 기사 |
| 정리 | 규칙성 | 집중력 | 관리자, 계획가 |
| 음악/일기 | 감성 | 창의력 | 치유사, 시인 |
| 휴식 | 체력 | 스트레스 감소 | 균형형, 명상가 |

## 5. 최종 성장 타입 (강아지 의인화)

| 타입 | 조건 | 강아지 포즈 | 다음 시즌 보너스 |
|---|---|---|---|
| 똑똑한 학자견 | 지식 높음 + 집중력 높음 | 안경 쓴 강아지, 책 들기 | 집중 태그 보상 +15% |
| 창의력 많은 예술견 | 창의력 높음 + 감성 높음 | 팔레트와 붓 들기, 상상의 표정 | 창작 표현 더 풍부하게 |
| 균형잡힌 왕견 | 모든 능력 균형 + 스트레스 낮음 | 왕관 쓴 강아지, 우아한 자세 | 모든 태그 보상 +20%, 피로 -50% |
| 밤샤수견 | 밤 집중 많음 + 지식 높음 + 스트레스 높음 | 졸린 눈, 별을 보는 포즈 | 밤 집중 보상 +20%, 피로 위험 증가 |
| 활발한 탐험견 | 체력 높음 + 규칙성 높음 | 가방 메고, 신나는 표정, 꼬리 흔들기 | 운동 태그 보상 +15% |
| 따뜻한 치유견 | 감성 높음 + 체력 높음 | 포근한 포즈, 친구와 함께 | 휴식 태그 보상 +12%, 친구 호감 +30% |
| 끈기있는 학자견 | 집중력 높음 + 지식 높음 + 중간 스트레스 | 진지한 표정, 책 정렬하는 포즈 | 집중 완료 보상 +15%, 효율성 +10% |
| 자유로운 몽상견 | 창의력 높음 + 규칙성 낮음 | 멍한 표정, 구름 바라보기 | 창의력 이벤트 +30%, 랜덤 만남 증가 |

## 6. MVP 추천 성장 타입 (강아지 의인화)

초기 MVP에서는 성장 타입을 6개로 시작한다. (강아지 의인화)

1. 똑똑한 학자견 (지식/집중력 높음)
2. 창의력 많은 예술견 (창의력/감성 높음)
3. 활발한 탐험견 (체력/규칙성 높음)
4. 따뜻한 치유견 (감성/체력 높음)
5. 균형잡힌 왕견 (모든 능력 균형)
6. 밤샤수견 (밤 집중 + 스트레스 높음)

이 6개만으로도 "내 집중 습관에 따라 강아지가 달라진다"는 핵심 재미를 만들 수 있다.

**Season 2 확장 계획:**
- 새로운 강아지 타입 추가 (끈기있는 학자견, 자유로운 몽상견)
- 고양이 캐릭터 추가 (새로운 의인화 동물)

## 7. 시즌 점수 계산 방식

앱 화면에서는 복잡한 수식을 보여주지 않고, 내부 계산만 사용한다.

```text
완료한 집중 시간 = 기본 성장 경험치
집중 품질 = 능력치 보정
태그 = 성장 방향 결정
휴식 = 스트레스 조절
연속 출석 = 규칙성 증가
중단/이탈 = 스트레스 증가
```

예시:

```text
공부 태그 8회
창작 태그 3회
휴식 부족
밤 집중 많음

결과:
지식 + 집중력 높음
스트레스 높음
=> 밤샘 연구자 후보
```

## 8. 시즌 종료 결과 화면

14일차가 끝나면 사용자는 다음 정보를 받는다.

- 이번 시즌 대표 성장 타입
- 가장 많이 사용한 집중 태그
- 가장 많이 오른 능력치 2개
- 위험했던 능력치 1개
- 캐릭터의 새로운 표정, 행동, 액세서리
- 다음 시즌에 적용되는 작은 보너스

결과 화면 예시:

```text
이번 시즌 콩냥이는 "밤샘 연구자"로 성장했어요.

주요 습관:
- 공부 집중이 많았어요.
- 밤 시간 집중이 강했어요.
- 휴식이 부족해서 스트레스가 높아졌어요.

다음 시즌 효과:
- 밤 집중 보상 +10%
- 피로 누적 속도 +5%
```

## 9. 기획 우선순위 (태그 4개 기준)

구현 우선순위는 다음 순서가 적합하다. (4태그 단순화 반영)

| 순서 | 항목 | 의존성 | 난이도 | 예상 기간 |
|------|------|--------|--------|----------|
| 1 | 태그 UI (집중/운동/휴식/잠) | - | ⭐ | 1주 |

| 4 | 극단화 감지 + 다양성 지표 | 3 | ⭐ | 3-5일 |
| 5 | 타입 결정 알고리즘 | 4 | ⭐⭐ | 3-5일 |

**병렬 작업 가능한 부분:**
- Phase 1 (1-3번): BE 팀이 병렬로 진행 가능
- Phase 3 (6-7번): Phase 2 완료 후 병렬 진행 가능

## 10. 태그 단순화 및 기준값 확정 (게임 기획자 버전)

프린세스 메이커 2를 참고하여, **본 기능에 충실하면서 극단화를 방지하는 4태그 시스템**으로 단순화한다.

### 핵심 설계 원칙

1. **태그 4가지로 단순화** (본 기능 중심)
   - 집중 (공부/업무)
   - 운동
   - 휴식
   - 잠 (시간 없이 카운트만)

2. **극단화 방지 메커니즘**
   - 한 태그만 극단화 → 번아웃 (약한 결과)
   - 다양한 태그 사용 → 프린세스 (최고 결과)
   - 프린세스 메이커 2 원리 적용: 균형이 최고의 보상

3. **사용자 경험**
   - 7일차 중간 평가: 캐릭터 표정 + 말풍선으로 경고
   - 14일 결과: 명확한 피드백과 차등 보너스

---

### 1단계: 능력치 점수 계산 (0~100)

**4가지 태그 기반 능력치:**

```
지식 = (집중 태그 횟수 × 6)
집중력 = (집중 태그 완료 × 4) + (중단 × -8)
체력 = (운동 태그 × 8) + (휴식 태그 × 3) + (잠 × 2)
감성 = (휴식 태그 × 5) + (잠 × 2)
규칙성 = (연속 출석 일수 × 3)
스트레스 = (중단 × 5) - (휴식 × 2) - (잠 × 1)
```

**예시 (14일 플레이):**
```
집중 8회 → 지식 48, 집중력 32
운동 4회 → 체력 32
휴식 6회 → 체력 18, 감성 30, 스트레스 -12
잠 7회 → 체력 14, 감성 14, 스트레스 -7
연속 출석 14일 → 규칙성 42
중단 1회 → 집중력 -8, 스트레스 +5
```

---

### 2단계: 극단화 감지 및 페널티

**극단화 감지 조건 (다음 중 하나):**

```
1. 한 능력치만 70+ AND 나머지 모두 30 이하
2. 한 능력치만 75+ AND 나머지 하나만 40-50, 나머지는 30 이하

극단화 페널티:
- 최고 능력치 -30
- 스트레스 +25 (번아웃 강화)
```

**극단화 예시:**
```
집중만 20회 → 지식 100
나머지 태그 1-2회만 → 체력 5, 감성 7 등 극도로 낮음

극단화 감지: 지식 100 + 나머지 모두 <40
페널티 적용:
- 지식: 100 - 30 = 70
- 스트레스: +25
→ 결과: "번아웃" (최악 등급)
```

---

### 3단계: 다양성 지표 (4가지 태그)

```
태그 다양성 계산:
- 1-2개 태그만: "편중" (-15 점수)
- 3개 태그: "표준" (±0)
- 4개 태그 모두: "전방위" (+20 점수)

보너스 적용:
- 전방위: 모든 능력치 +20
- 편중: 최고 능력치 -15 추가 감소
```

---

### 4단계: 타입 결정 기준 (최종)

| 타입 | 능력치 조건 | 다양성 | 등급 |
|------|-----------|--------|------|
| **프린세스** ⭐⭐⭐⭐⭐ | 모든 능력 ≥50, 스트레스 <20 | 4개 태그 모두 | 최고 |
| **조화로운 현인** ⭐⭐⭐⭐ | 3개 이상 능력 ≥45, 스트레스 <30 | 3-4개 태그 | 높음 |
| **학자** ⭐⭐⭐ | 지식 ≥60, 집중력 ≥50 | 2개 이상 | 중상 |
| **스포츠맨** ⭐⭐⭐ | 체력 ≥65, 규칙성 ≥50 | 2개 이상 | 중상 |
| **치유사** ⭐⭐⭐ | 감성 ≥60, 체력 ≥45 | 2개 이상 | 중상 |
| **번아웃** ⭐ | 한 능력 ≥70 + 나머지 ≤30, 스트레스 ≥70 | 1개 태그만 | 최악 |

---

### 5단계: 캐릭터 표정 & 말풍선 (7일차 중간 평가)

**7일차 중간 평가 UI (캐릭터 표정 + 말풍선):**

```
상태 1: 균형잘잡은 플레이 (4개 태그 모두 1회 이상)
┌─ 캐릭터 표정: 밝은 미소, 반짝이는 눈, 활기찬 포즈
├─ 말풍선: "오, 다양한 활동을 하고 있네! 이 정도면 좋은 방향이야~"
└─ 피드백: 계속 이 정도면 좋은 결과를 볼 수 있어!

상태 2: 한 태그 지나치게 많음 (1개만 7회 이상, 나머지 1회 이하)
┌─ 캐릭터 표정: 피곤한 눈, 약간 처진 입, 늘어진 어깨
├─ 말풍선: "음... 계속 이것만 하는군요? 😓 다른 활동도 해보면 어떨까요?"
└─ 경고: ⚠️ 현재 궤적은 번아웃입니다
   → 권장: 운동, 휴식, 잠 같은 다른 활동도 시도해보세요!

상태 3: 거의 활동 없음 (집중 3회 이하)
┌─ 캐릭터 표정: 졸린 눈, 흐린 표정, 축 늘어진 모습
├─ 말풍선: "...뭔가 일어나고 있는 건가요? 😴"
└─ 격려: 작은 집중부터 시작해보세요!

    'burnout': { focus: 0.5, exercise: 0.5, rest: 0.5, sleep: 0.5 },
    'scholar': { focus: 1.15, exercise: 1.0, rest: 1.0, sleep: 1.0 },
    // ...
  }
  return bonusMap[lastCharacterType]
}
```

## 10. 핵심 차별점

일반적인 포모도로 캐릭터 앱은 집중 시간에 비례해서 캐릭터가 자란다. Pomochi의 차별점은 사용자의 습관 방향이 캐릭터의 성격과 미래를 바꾼다는 점이다.

```text
Forest: 집중하면 나무가 자람
Finch: 자기관리하면 펫이 성장함
Habitica: 할 일을 RPG 보상으로 바꿈
Pomochi: 집중 습관이 캐릭터의 인생 경로를 바꿈
```

따라서 Pomochi의 장기 방향은 "포모도로 기반 성장 시뮬레이션"이다.
